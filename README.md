# Flask Budd - Personal Finance Tracker ... with Flask

A Flask web application for tracking personal finances with an MS-DOS inspired interface.

## Features
- Add income and expense transactions
- Filter transactions by category, type, and date range
- View account balance and transaction history
- Keyboard-driven navigation

## Setup
```bash
pip install -r requirements.txt
python app.py
```

Visit `http://localhost:5000`

## Usage
- Navigate using keyboard shortcuts (Alt+A for Add Transaction, etc.)
- Filter transactions using the search form
- All data stored in SQLite database
